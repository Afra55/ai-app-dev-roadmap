## 任务清单

- [ ] 每份问答文档自测 3 题，能结合自己仓库回答
- [ ] 准备 1 分钟 / 3 分钟自我介绍（见 [mock-interview.md](mock-interview.md)）
- [ ] 完成 1 次模拟面试（朋友 / 录音自评 / AI 模拟）
- [ ] 准备 2 个「项目难点 + 如何解决」故事

---

## 答题结构（推荐）

1. **定义**：一句话概念
2. **项目实践**：「我在 xxx 里是这样做的」+ 文件路径
3. **权衡/改进**：当前局限与下一步优化
