## 失败场景应答

| 问题 | 参考答法 |
|------|----------|
| 检索结果为空 | 检查是否 ingest；调 chunk；Prompt 要求明确说「未找到」 |
| API 超时 | 重试、降级端侧、缓存 FAQ |
| Agent 乱调工具 | 优化 tool 描述；简单问题走路由 bypass Agent |
| 模型幻觉 | 强化 RAG 约束 + 来源引用 + 降低 temperature |

## 行为面常见问题

1. **为什么转 AI？** — Android 经验 + 端侧 AI 趋势 + 已完成系统学习与项目
2. **最大的学习困难？** — 例：跨周模块导入、Embedding 首次下载；如何通过包结构与 `verify_setup` 解决
3. **团队协作？** — 结合过往 Android 项目
