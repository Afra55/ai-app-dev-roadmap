## 公司调研清单（每家 10 分钟）

- [ ] 业务是否用大模型 / 知识库 / 移动端 AI？
- [ ] 技术栈偏 Python 还是 Android？
- [ ] 国企/银行是否重合规、审计？
- [ ] 岗位更偏研发还是解决方案？

## 投递渠道

| 渠道 | 说明 |
|------|------|
| 官网 / 内推 | 优先，回复率更高 |
| Boss / 猎聘 / 智联 | 注意 JD 真实性 |
| 国企官网 | 周期 long，提前投递 |
| GitHub / 技术社区 | 展示仓库与 Demo 链接 |

---

## 避免事项

- 海投同一版简历到差异很大的岗位
- 简历项目与 GitHub 仓库对不上
- 面试时演示从未跑过的命令
- 对银行/金融项目不说明「教学演示」性质
